# کیبورد ابهام (Abham Keyboard)

این یک پروژه‌ی کامل اپلیکیشن اندرویدی است: یک کیبورد ساده‌ی فارسی + انگلیسی با فونت خوش‌نویس.

## چی توی این پروژه هست؟

- صفحه‌ی اصلی اپ با عنوان «کیبورد ابهام»
- خود کیبورد: حروف فارسی، حروف انگلیسی (با دکمه تغییر زبان)، Space، Backspace، Enter
- فونت سفارشی روی همه‌ی دکمه‌ها

## قبل از ساختن (Build): یک قدم ضروری

پوشه‌ی فونت (`app/src/main/res/font/`) فعلاً خالی است و باید یک فایل فونت
به اسم دقیق `abham_script.ttf` در آن قرار بدهی. فایل README.txt داخل همان
پوشه توضیح می‌دهد از کجا یک فونت فارسی رایگان و قانونی (Vazirmatn) دانلود کنی.
هر فونت دیگری هم که دوست داشتی، کافیست اسمش را به همین شکل تغییر بدهی.

## چطور بسازم (Build) و نصب کنم؟

### روش پیشنهادی: با Android Studio روی کامپیوتر
۱. Android Studio را روی کامپیوتر نصب کن (رایگان است: developer.android.com/studio)
۲. این پوشه‌ی پروژه را با Android Studio باز کن (Open an existing project)
۳. صبر کن تا Gradle Sync تمام شود (بار اول کمی طول می‌کشد)
۴. فایل فونت را طبق توضیحات بالا اضافه کن
۵. از منو: Build > Build Bundle(s) / APK(s) > Build APK(s)
۶. فایل APK ساخته‌شده را روی گوشی کپی و نصب کن

### روش پیشرفته: ساختن مستقیم در Termux
این روش به دلیل محدودیت حافظه روی گوشی‌های معمولی پیشنهاد نمی‌شود مگر گوشی
حداقل ۶ گیگابایت رم آزاد داشته باشد. نیاز است Android SDK + Gradle + JDK
به‌صورت کامل نصب شوند که ممکن است زمان‌بر و پرریسک باشد.

## بعد از نصب APK، چطور کیبورد را فعال کنم؟

۱. اپ «کیبورد ابهام» را باز کن
۲. روی دکمه «باز کردن تنظیمات کیبورد» بزن
۳. در صفحه‌ی تنظیمات اندروید، کنار «کیبورد ابهام» را روشن کن
۴. هنگام تایپ، روی آیکون تغییر کیبورد (پایین صفحه، کنار صفحه‌کلید) بزن و
   «کیبورد ابهام» را انتخاب کن

## محدودیت‌های این نسخه (قابل توسعه در آینده)

- بدون پیشنهاد کلمه (autocomplete) یا تصحیح خودکار
- بدون صفحه‌ی اعداد و نشانه‌ها (می‌توان بعداً اضافه کرد)
- چیدمان حروف ساده است، نه دقیقاً مثل کیبورد رسمی گوگل
